Docker ile Kurulum

Projeyi GitHub'dan indirin ve tek komutla tüm servisleri başlatın.

1. Projeyi İndirin
bash
git clone https://github.com/mcemkoca/Yafes_Pars.git
cd Yafes_Pars
2. Ortam Dosyasını Oluşturun

Proje kök dizininde .env adında bir dosya oluşturun:

.env
# SQL Server yönetici şifresi — en az 8 karakter, büyük/küçük harf + rakam
SA_PASSWORD=YafesStr0ng!2024

# JWT — geliştirme ortamında boş bırakın
AUTH_AUTHORITY=
AUTH_AUDIENCE=
🔐
SA_PASSWORD en az 8 karakter olmalı ve büyük harf, küçük harf, rakam içermelidir. Aksi hâlde SQL Server başlamaz.
3. Servisleri Başlatın
bash
docker compose up -d
Beklenen Çıktı 
[+] Running 2/2
 ✔  Container yafes-sql   Started
 ✔  Container yafes-api   Started

İlk başlatmada SQL Server imajı indirilir (~1.5 GB). Bu işlem birkaç dakika sürebilir.

4. Container Durumunu Kontrol Edin
bash
docker compose ps
Beklenen Çıktı 
NAME          STATUS          PORTS
yafes-sql     Up (healthy)    0.0.0.0:1433->1433/tcp
yafes-api     Up              0.0.0.0:5000->8080/tcp
ℹ️
Her iki container da Up durumunda olmalıdır. SQL container'ın healthy göstermesi 30–60 saniye sürebilir.
Adım 3 / 6

Veritabanı Kurulumu

21 migration dosyasını sırayla çalıştırarak veritabanını oluşturun.

ℹ️
Migration'lar idempotenttir — hata olmadığı sürece tekrar çalıştırılabilir. Sıralama önemlidir: 000 → 020 sırasıyla çalıştırın.
Seçenek A — SSMS ile (Önerilen) Görsel
  1. SSMS'i açın
  2. Connect → Server name: localhost,1433
  3. Authentication: SQL Server Authentication → Login: sa / Şifre: .env'deki SA_PASSWORD
  4. File → Open ile database/migrations/000__create_database.sql dosyasını açın
  5. F5 ile çalıştırın
  6. 000 → 020 tüm dosyaları sırayla tekrarlayın (21 dosya)
Microsoft SQL Server Management Studio
-- 000__create_database.sql
CREATE DATABASE [YafesPars]
COLLATE SQL_Latin1_General_CP1_CI_AS;
GO

-- Migration tamamlandı
PRINT 'Database created.'
Seçenek B — sqlcmd ile (Otomasyon) Komut satırı
PowerShell
# Proje kök dizininde çalıştırın
Get-ChildItem database\migrations\*.sql | Sort-Object Name | ForEach-Object {
    Write-Host "Running: $($_.Name)" -ForegroundColor Cyan
    sqlcmd -S localhost,1433 -U sa -P "YafesStr0ng!2024" \
           -d master -i "$($_.FullName)"
}
⚠️
SA_PASSWORD değerini kendi .env dosyanızdaki değerle değiştirin.
