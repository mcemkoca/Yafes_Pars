SQL Server Installeren

Install SQL Server 2022 Express + SQL Server Management Studio

📝 Systeemvereisten / Requirements

  • Windows Server 2012+ of Windows 10/11
  • SQL Server 2016+ (aanbevolen: SQL Server 2022 Express)
  • Minimaal 4 GB RAM, 10 GB vrije schijfruimte
  • Administratieve rechten op de server

⬇ Download & Installatie

Download SQL Server 2022 Express en SQL Server Management Studio (SSMS) van de officiële Microsoft website:

💻 SQL Server 2022 Express Setup
🗃 SQL Server Installation Center
Kies "New SQL Server stand-alone installation"
Stap 1: Start het setup programma
🔧 Installation Type
✅ Feature Selection
Selecteer: Database Engine Services + SQL Client Connectivity SDK
Stap 2: Selecteer features
🔐 Instance Configuration
🔧 Instance: SQLEXPRESS
Authentication: Mixed Mode (SQL + Windows)
Stap 3: Configureer instance & authenticatie

⚙ Stille Installatie (Unattended)

Gebruik een configuratiebestand voor geautomatiseerde installatie:

PowerShell - Unattended Install 
# Start unattended SQL Server 2022 Express installatie
SQLEXPR_x64_ENU.exe /ConfigurationFile=sql\ConfigurationFile.ini /IACCEPTSQLSERVERLICENSETERMS

# Of met parameters direct:
SQLEXPR_x64_ENU.exe /ACTION=Install /FEATURES=SQLENGINE /INSTANCENAME=SQLEXPRESS /SQLSVCACCOUNT="NT AUTHORITY\NETWORK SERVICE" /SECURITYMODE=SQL /SAPWD="YourStrongPassword123!" /TCPENABLED=1
💡
Tip: Zorg dat TCP/IP ingeschakeld is in SQL Server Configuration Manager voor netwerktoegang. Ensure TCP/IP is enabled in SQL Server Configuration Manager for network access.

Database Deployen

Deploy the database using PowerShell or SSMS

Kies een van de twee methoden om de AssureManager database te deployen:

⚡ Optie A: PowerShell (Aanbevolen)

Automatisch alle 8 scripts uitvoeren met een enkel commando. Snel en foutloos.

Automatisch Snel

💻 Optie B: SSMS (Handmatig)

Voer 8 SQL scripts handmatig uit in de juiste volgorde via SQL Server Management Studio.

Handmatig Controle

⚡ Optie A: PowerShell Script

PowerShell - .\sql\deploy.ps1 
# Open PowerShell als Administrator en navigeer naar project root
cd C:\AssureManager

# Voer het deployment script uit
.\sql\deploy.ps1

# Het script voert automatisch uit:
# 1. 01_create_database.sql      - Database aanmaken
# 2. 02_schema.sql               - 90 tabellen aanmaken
# 3. 03_constraints.sql          - 36 FK + 3 UK + 2 CK constraints
# 4. 04_seeds.sql                - 2.000+ lookup records
# 5. 05_triggers.sql             - 13 triggers (timestamps + business rules)
# 6. 06_stored_procedures.sql    - 18 stored procedures
# 7. 07_views.sql                - 6 reporting views
# 8. 08_test_data.sql            - 300 realistische testrecords

💻 Optie B: SSMS Handmatig

Voer de volgende scripts in exacte volgorde uit in SSMS:

SQL Script Volgorde (8 scripts) 
-- Script 01: Database aanmaken
:r .\01_create_database.sql
GO

-- Script 02: Schema - Alle 90 tabellen aanmaken
:r .\02_schema.sql
GO

-- Script 03: Constraints - 36 Foreign Keys + 3 Unique + 2 Check
:r .\03_constraints.sql
GO

-- Script 04: Seeds - 2.000+ lookup records (landen, talen, domeinen, ...)
:r .\04_seeds.sql
GO

-- Script 05: Triggers - 13 triggers voor timestamps en business rules
:r .\05_triggers.sql
GO

-- Script 06: Stored Procedures - 18 SPs voor complexe queries
:r .\06_stored_procedures.sql
GO

-- Script 07: Views - 6 reporting views
:r .\07_views.sql
GO

-- Script 08: Test Data - 300 realistische testrecords
:r .\08_test_data.sql
GO
⚠
Belangrijk: Draai scripts ALTIJD in volgorde 01 → 08. Foreign Key errors ontstaan bij verkeerde volgorde! Always run scripts in order 01 → 08 to avoid FK errors.

✅ Verificatie

SQL - Verificatie Query 
USE AssureManagerDB;
SELECT
  (SELECT COUNT(*) FROM sys.tables) AS 'Tabellen',
  (SELECT COUNT(*) FROM sys.procedures) AS 'Stored Procedures',
  (SELECT COUNT(*) FROM sys.views) AS 'Views',
  (SELECT COUNT(*) FROM sys.triggers) AS 'Triggers';

-- Verwacht resultaat: 90 | 18 | 6 | 13

API Starten

Start the Node.js/Express REST API server

🚀 Quick Start

Terminal 
# Navigeer naar de API directory
cd api

# Installeer dependencies (eerste keer)
npm install

# Start de server (development mode met auto-reload)
npm start

# Output: Server listening on port 3001
#         Connected to SQL Server (AssureManagerDB)
#         API ready at http://localhost:3001/api/v1

🔐 Environment Variables (.env)

Maak een .env bestand aan in de api/ folder:

api/.env 
# Server configuratie
PORT=3001
NODE_ENV=development

# Database configuratie (SQL Server)
DB_HOST=localhost
DB_PORT=1433
DB_NAME=AssureManagerDB
DB_USER=sa
DB_PASSWORD=YourStrongPassword123!

# JWT Secret voor authenticatie
JWT_SECRET=your-jwt-secret-key-change-in-production

# CORS (voor frontend)
CORS_ORIGIN=http://localhost:3000

VM Deployment (Optioneel)

Automated Hyper-V/QEMU VM deployment for production

Deze stap is optioneel. Gebruik het voor volledig geautomatiseerde VM deployment:

🖨 VM Aanmaken

PowerShell - Setup-VM.ps1 
# Voer het master VM setup script uit
.\Setup-VM.ps1

# Het script voert uit:
# 1. Download Windows Server 2022 ISO
# 2. Maak VHDX virtuele harde schijf (80GB)
# 3. Genereer autounattend.xml (unattended install)
# 4. Mount ISO + VHDX
# 5. Kopieer AssureManager bestanden
# 6. Start VM in Hyper-V / QEMU

📑 Autounattend.xml

Het antwoordbestand configureert Windows automatisch zonder gebruikersinteractie:

autounattend.xml (fragment) 
<settings pass="windowsPE">
  <component name="Microsoft-Windows-Setup">
    <UserData>
      <ProductKey>
        <Key>D7643-K2T7V-XXXXX-XXXXX-XXXXX</Key>
      </ProductKey>
    </UserData>
    <ImageInstall>
      <OSImage>
        <InstallFrom>
          <MetaData wcm:action="add">
            <Key>/IMAGE/NAME</Key>
            <Value>Windows Server 2022 Standard</Value>
          </MetaData>
        </InstallFrom>
      </OSImage>
    </ImageInstall>
  </component>
</settings>
💡
Tip: Pas de autounattend.xml aan voor jouw omgeving (product key, admin wachtwoord, netwerkinstellingen). Customize the autounattend.xml for your environment.

💡 Data Flow

1. Gebruiker opent Dashboard

React 19 app wordt geladen in de browser. KPI cards laden automatisch via /api/v1/dashboard/stats.

2. API Verwerkt Request

Express ontvangt HTTP request, voert SQL query uit via mssql package, retourneert JSON.

3. SQL Server Retourneert Data

Stored procedures of directe queries op 90+ tabellen. Triggers loggen wijzigingen automatisch.

4. UI Toont Resultaat

React rendert data in tabellen, cards en grafieken. Toast toont status van elke actie.

Troubleshooting

Veelvoorkomende problemen en oplossingen
ProbleemOplossing
sqlcmd not found Installeer SQL Server Command Line Utilities van Microsoft. Voeg C:\Program Files\Microsoft SQL Server\Client SDK\ODBC\170\Tools\Binn\ toe aan PATH.
Connection failed Controleer: 1) SQL Server service draait Get-Service MSSQL$SQLEXPRESS, 2) TCP/IP is ingeschakeld in SQL Server Configuration Manager, 3) Firewall poort 1433 is open.
FK error Draai scripts in de correcte volgorde: 01_create_database.sql t/m 08_test_data.sql. Foreign keys vereisen dat referenced tabellen eerst bestaan.
Port 3001 in use Wijzig PORT=3001 naar een ander poortnummer in het .env bestand. Herstart de API server.
npm install faalt Update Node.js naar versie 20+. Verwijder node_modules en package-lock.json, voer npm install opnieuw uit.
CORS error in browser Controleer CORS_ORIGIN in .env. Zorg dat deze matcht met de frontend URL (meestal http://localhost:3000).
Database timeout Verhoog de timeout in de DB connectie string. Controleer netwerkverbinding. Zorg dat SQL Server Browser service draait.
VM start niet op Controleer of Hyper-V ingeschakeld is: Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V. Zorg voor voldoende RAM (min. 4GB).
