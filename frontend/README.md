# Yafes Pars Frontend

Next.js admin/broker panel foundation for Yafes Pars.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- TanStack Query
- lucide-react icons

## Configuration

Set the API base URL with:

```powershell
$env:NEXT_PUBLIC_API_BASE_URL="https://localhost:5001"
```

No backend secrets are stored in the frontend.

## Commands

```powershell
npm install
npm run typecheck
npm run lint
npm run build
npm run dev
```
